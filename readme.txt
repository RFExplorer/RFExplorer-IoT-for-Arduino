RF Explorer 3G+ IoT for Arduino DUE
-----------------------------------

For more details and reference: http://www.rf-explorer.com/IoT

For examples and library description: https://github.com/RFExplorer/RFExplorer-IoT-for-Arduino/wiki

For contact and support: http://www.rf-explorer.com/contact

--- DESCRIPTION ---

RF Explorer 3G+ IoT is a customizable Spectrum Analyzer for Arduino / Genuino / Seeeduino boards.

This library includes examples and source code to get started on using RF Explorer 3G+ IoT board in your Arduino DUE.
  
--- USE --- 

For details on how to install the library and run the examples, please visit www.rf-explorer.com/IoT in the Arduino section.
